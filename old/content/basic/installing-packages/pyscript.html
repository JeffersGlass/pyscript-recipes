To install a new Python package in PyScript, include the name of the package or the URL of the self-built wheel in the packages list in code <py-config>:

{{< highlight html "linenos=True">}} packages = ['astropy', 'pyyaml', 'https://example.com/files/someWheel-py3-none-any.whl'] # packages can be from PyPI, Pyodide Packages, or self-built wheels {{< /highlight >}}
