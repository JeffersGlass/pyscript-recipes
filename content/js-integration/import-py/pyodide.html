--- title: "Pyodide" ---

Once Pyodide has been loaded, we can extract objects from its global scope. If we have objects in the Python global scope, we can extract them in JavaScript using pyodide.globals.get(...):

{{< highlight js "linenos=True">}} async function main() { let pyodide = await loadPyodide(); // Pyodide is now ready to use... console.log(pyodide.runPython(` my_string = "Hello again" `)); const s = pyodide.globals.get('my_string') console.log(s) #logs hello again }; main(); {{< /highlight >}}
