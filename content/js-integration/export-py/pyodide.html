--- title: "Pyodide" ---

Once Pyodide has been loaded, we can use JavaScript to directly create Python objects using pyodide.globals.set(name, value):

{{< highlight js "linenos=True">}} async function main() { let pyodide = await loadPyodide(); const s = 'hello world' pyodide.globals.set('my_string', s) // Pyodide is now ready to use... pyodide.runPython(` print(my_string) `); }; main(); {{< /highlight >}}
