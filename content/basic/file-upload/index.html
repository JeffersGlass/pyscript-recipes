--- Title: "File Upload" flavor: 'basic' weight: 3 ---
This page is about setting up an interface for the users of your page to upload files. To setup files to be loaded on page load, use <py-config> [[fetch]]...

The PyScript and Pyodide versions of this tutorial are identical.

Purpose

As front-end frameworks, it is often useful for users of PyScript or Pyodide to allow users to upload their own individual files to the web page for processing. These might be CSV or parquet files of data, images for processing, text files etc.
